## About The Project

Product docker-compose of the strapi.

## Built With

* [Strapi](https://strapi.io/)
* [Postgres](https://www.postgresql.org/)
* [Nginx](https://www.nginx.com/)

## Install

1. Pull image

   ```sh
   docker-compose pull
   ```

2. Start

   ```sh
   docker-compose up -d
   ```
